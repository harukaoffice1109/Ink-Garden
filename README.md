# 墨庭 Ink Garden

原创 WebGL 流墨生境互动艺术。项目是纯前端静态站，不需要后端、数据库或构建步骤。

## 功能

- WebGL2 GPU 流体水墨
- 墨黑、靛蓝、朱砂、松绿四种墨色
- 墨刷、墨滴、漩涡、清水、落花、召鱼工具
- 金鱼巡游、避让、靠近指尖、摆尾扰墨
- 花瓣、水纹、旧墨记忆和自动静观模式
- Web Audio 程序生成琴音、水声和交互音
- 保存 PNG、录制 WebM、复制分享链接
- 移动端自适应和低性能设备降级

## 本地运行

```bash
npm run start
```

然后打开：

```text
http://localhost:4173
```

## 检查

```bash
npm run check
```

## 免费部署

这是静态站，可以直接部署到 Cloudflare Pages、GitHub Pages、Netlify 或 Vercel。
部署目录就是项目根目录。
